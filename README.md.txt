## Objective
To detect spam emails using machine learning techniques.

## Description
This project uses Python and Machine Learning to classify emails as Spam or Not Spam.
Naive Bayes algorithm is used for classification.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn

## Dataset
SMS Spam Collection Dataset.

## How to Run the Project
1. Install required libraries:
   pip install -r requirements.txt
2. Run the program:
   python main.py

## Output
The system predicts whether an email message is Spam or Not Spam.
